Converts a same-material characterization batch into a reviewable report: dominant peak stability, FWHM trend, Scherrer upper-bound size trend, fit-quality flags, and scientist-review figures in one portable HTML memo.

What a buyer gets

  • 1
    Raw data intake

    Parses RRUFF-format .txt files with variable-length metadata headers. Reads wavelength from file; defaults to Cu Kα (1.54056 Å).

  • 2
    Repeatable peak detection

    Difference of Gaussians (after xrfit): a narrow Gaussian preserves peaks while a wide one approximates baseline. Peak candidates are selected from an adaptive MAD-noise threshold, reducing hand-tuned settings across repeat batches.

  • 3
    Scientist-reviewable fits

    Per-peak ROI lmfit PseudoVoigt + linear background. Quality filters reject weak or poor fits, and each sample gets an overlay and residual panel for human review.

  • 4
    Process signal extraction

    Same-material batch: tracks dominant peak position, FWHM, d-spacing, and Scherrer upper-bound size so a process engineer sees whether the batch drifted, broadened, or needs review.

  • 5
    Decision memo

    Jinja2 HTML memo with process signal, decision implication, validation notes, summary table, full per-peak table, and base64-embedded figures. Single portable file.

The commercial gap

Peak fitting is not the whole job. Labs still need the last mile: repeatable batch analysis, validation notes, and a memo a process engineer or technical manager can act on without rebuilding the analysis in a spreadsheet. This demo shows the kind of bespoke automation I build for characterization teams: the scientific method stays visible, but the reporting burden is automated.

Technical approach

Peak detection

Difference of Gaussians + adaptive MAD-noise threshold for repeatable candidate detection across similar patterns.

Peak fitting

lmfit PseudoVoigt (Gaussian + Lorentzian mix), per-peak ROI ±0.25°. Levenberg-Marquardt with linear local background. ~30 ms per peak.

Crystallite size

Scherrer equation D = Kλ/βcosθ, K=0.9. Upper-bound estimates; instrument broadening not subtracted.

Outlier flagging

R² threshold (poor fit) + IQR on FWHM and crystallite size. Single Flag column with R² value embedded — no false confidence at small n.

Output

Single-file HTML memo: process signal, decision implication, validation notes, summary table, full per-peak table, and sample fits.
